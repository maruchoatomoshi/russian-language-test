# Электронный тест по русскому языку

Отдельный статический проект для публикации через GitHub Pages. Не связан с ZHIDAO Protocol.

## Что внутри

- `index.html` - интерфейс теста.
- `questions.js` - 100 вопросов, варианты, правильные ответы и объяснения.
- `answer_key.csv` - ключ ответов для преподавателя.
- `corrections_report.md` - краткий отчёт о правках спорных вопросов.

## Как работает

1. Студент вводит имя и группу.
2. Отвечает на 100 вопросов.
3. После завершения видит баллы, процент, оценку и разбор.
4. В разборе выбранный неправильный вариант отмечается красным, правильный ответ - зелёным.
5. Для неправильных ответов показывается объяснение.
6. Результат можно скопировать, скачать как CSV или сохранить в PDF через печать.

## Публикация на GitHub Pages

1. Откройте `Settings -> Pages`.
2. Выберите `Deploy from a branch`.
3. Branch: `main`, folder: `/root`.
4. Нажмите `Save`.
5. После публикации ссылка будет вида `https://USERNAME.github.io/russian-language-test/`.

## Важно

GitHub Pages хранит только статические файлы. Он не сохраняет результаты студентов на сервере. Для автоматического сбора результатов позже можно подключить Google Sheets, Supabase, Firebase или Telegram-бота.
